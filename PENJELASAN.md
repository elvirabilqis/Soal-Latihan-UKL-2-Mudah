Program ini digunakan untuk menentukan apakah suatu bilangan merupakan ganjil atau genap

Deskripsi Program : 
Program meminta pengguna untuk memasukkan sebuah bilangan bulat. Kemudian, program akan memeriksa apakah bilangan tersebut genap atau ganjil menggunakan operator modulus (`%`)

Cara Kerja Program :
1.  Program menampilkan pesan untuk meminta pengguna memasukkan sebuah bilangan
2.  Input dari pengguna dibaca menggunakan `Scanner`
3.  Program memeriksa :
   - Jika sisa pembagian bilangan dengan 2 (`bilangan % 2`) sama dengan 0, maka bilangan genap
   - Jika tidak sama dengan 0, maka bilangan ganjil
4.  Program menampilkan hasil di terminal
5.  `Scanner` ditutup menggunakan `input.close()`
